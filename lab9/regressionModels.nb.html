Vida acelerada

Sabemos que los modelos de vida acelerada est√°n dados por el siguiente modelo:

\[T_{i} = \frac{T_0}{\varphi(X,\theta)}.\]

Funciones de riego para modelos Weibull, LogLogistico como tambi√©n funciones de supervivencia.

  • Weibull: \[ \]
#--- MODELOS DE REGRESION ---

#-Modelo de vida acelerada-
#Weibull
hw<-function(t,x,alpha,lam,theta){
  out<-exp(alpha*sum(x*theta))*lam*alpha*t^(alpha-1)
  out
}
#Loglogistico
hll<-function(t,x,alpha,lam,theta){
  out<-exp(alpha*sum(x*theta))*lam*alpha*t^(alpha-1)
  out<-out/(1+lam*exp(alpha*sum(x*theta))*t^alpha)
  out
}
sll<-function(t,x,alpha,lam,theta){
  out<-1/(1+lam*exp(alpha*sum(x*theta))*t^alpha)
  out
}
#Lognormal
sln<-function(t,x,mu0,b,theta){
  out<-1-pnorm((log(t)-mu0+sum(x*theta))/b)
  out
}

Mostremos el modelo de vida acelerada para el caso que \(X = (X_1, X_2)\) con \(X_i \sim \mathcal{Unif}(0,1)\).

NOTA:

  • La primer gr√°fica nos muestra c√≥mo crecen los riesgos. El individo base (x0) es la gr√°fica negra, el uno y el dos son el rojo y el verde, respectivamente.
  • La segunda gr√°fica nos muestra que el cociente de los riesgos es constante. Es importante que se sepa que solo passa esto para los modelos Weibull.

Ajustamos un modelo de vida acelerada es un modelo de regresi√≥n log-lineal, donde en este caso es: \(ln(T) = \alpha_{0} ln(wbc) + \alpha_{1} ln(wbc) + \alpha_{2}AG + \varepsilon\). En este caso \(\mu_{0} = \mathbb{E}[ln(T_{0})]\) y \(\varepsilon = ln(T) - \mu_{0}\) con media 0 y \(\varepsilon \perp \bf{x}\).

Ajuste de modelos

  • Weibull
  • Log Logistico
  • Log Normal
#Ajuste del modelo

xfitr3 <- survreg( Surv(Time,delta)~log(wbc)+AG,data=leucemia,dist="lognormal")

La interpretaci√≥n del modelo es el siguiente para el modelo Weibull:

  • Para la escala, no nos interesa ver su significancia, pues es solo un par√°metro de escala, no de localizaci√≥n. Adem√°s el iinter√©s es sobre 1 y no sobre 0 (su valor).

  • Para la variable dummie AG, elevamos a la \(e\) y restemos 1 para ver el porcentaje de cambio respecto si es 0 o 1. En este caso vemos que NA % con respecto al cambio de 0 a 1.

Interpretaci√≥n para el ‚Äúmejor‚Äù modelo:

  • El modelo lognormal presenta una log-verosimilitud menor que los dem√°s. Por tanto nos quedamos con dicho modelo.
