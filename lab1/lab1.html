Gráficas de funciones de densidad, riesgo y supervivencia

Geométrico

En este caso, la variable \(X \sim \text{Geom}(p)\) donde \(X \in \{0,1,...\}\). En clase se vió para \(X \in \{1,2,...\}\), en este caso, tendremos:

  • \(f_{X}(t) = p(1-p)^{t} \mathbb{I}_{\{0,1,...\}}\).
  • \(S_{X}(t) = \mathbb{P}[T > t] = (1-p)^{t+1} \quad t \in \{0,1,...\}\), aquí vemos que esta supervivencia evaluada en cero es distinta de 1!
  • \(h(t) = p\).
  • \(\kappa_{p} \leq \frac{ln(p)}{ln(1-p)} + 1\).

NOTA. En este caso, por errores de aritmética de punto flotante, la función \(h(t)\) no es cte por errores de redondeo al definir \(h(t)\).

#generar parametros
t<-0:15
p<-0.2
n<-length(t)
ft<-dgeom(t,p)
st<-1-pgeom(t,p)
ht<-ft
ht[2:n]<-ft[-1]/st[-n] # same as ft from 2 to N and st from 1 to N-1

# graph
par(mfrow=c(1,3))
plot(t,ft, col = 2, ylab = 'f', type = 'b')
plot(c(1,st), col = 4, ylab = 'St', type = 'b')
plot(t,ht, col = 6, ylab = 'ht', type = 'b')

Poisson

Notas - \(h(t)\) va creciendo, siempre, sin límite. Razón; \(f_{X}(t)\) siempre decrece con brincos positivos. - Para T’s grandes, hay errores de redondeo

Varias gráficas para: - \(\lambda = 0.5, 5, 10\)

for(TT in c(15)){
  for(lambda in c(0.5,5,10)){
    t<-0:TT #play with the value for t
    lam<-lambda
    n<-length(t)
    ft<-dpois(t,lam)
    st<-1-ppois(t,lam)
    ht<-ft
    ht[2:n]<-ft[-1]/st[-n]
    par(mfrow=c(1,3))
    plot(t,ft, col = lambda * 2, type = 'b', main = paste('Lambda = ', lambda))
    plot(t,st, col = lambda * 2, type = 'b')
    plot(t,ht, col = lambda * 2, type = 'b')
    
  }
  
}

Binomial negativo

Acuerdate que - \(f_{X}(t) = \binom{r+t-1}{t}p^{r}(1-p)^{t}\)

for(r in c(4,8,15)){
  for(p in c(1/4, 0.5, 7/8)){
  # times (t)
  t<-0:40
  n<-length(t)
  
  #distribution and functions
  ft<-dnbinom(t,r,p)
  st<-1-pnbinom(t,r,p)
  ht<-ft
  ht[2:n]<-ft[-1]/st[-n]
  
  #plot
  par(mfrow=c(1,3))
  plot(t,ft, col = p*8, type = 'b', main = paste('p = ', p,'y r = ',r))
  plot(t,st, col = p*8, type = 'b')
  plot(t,ht, col = p*8, type = 'b')
    
  }
}
